# Floré Haven

### Video Demo: <https://youtu.be/4-h4Frz0kfU?si=gAWN4gXH3JbBQuSd>

### Description:

Floré Haven is a full-stack e-commerce web application designed to deliver a refined and seamless online flower shopping experience. The platform simulates a real-world digital storefront where users can browse curated floral products, explore detailed descriptions, add items to a dynamic shopping cart, and complete a structured checkout process. The goal of this project was to combine elegant visual design with practical functionality to create a realistic and user-centered online store.

The idea behind Floré Haven is inspired by the emotional and aesthetic nature of flower shopping. Purchasing flowers is often connected to special occasions, celebrations, and meaningful moments. Therefore, the interface was carefully designed to feel soft, clean, and welcoming while maintaining clarity and ease of use. The layout focuses on visual hierarchy, spacing, and responsive design principles to ensure consistency across different screen sizes and devices.

From a technical perspective, the application follows a structured and modular architecture. The front-end is built using modern web technologies to create reusable components and maintain clean code organization. The shopping cart system dynamically updates as users add or remove products, demonstrating the use of state management and interactive UI behavior. Product listings are structured in a scalable way to allow easy expansion in the future.

Additionally, the project emphasizes user experience by ensuring intuitive navigation between pages, smooth transitions, and clear product presentation. Attention was given to responsiveness, accessibility, and overall performance to simulate a production-ready e-commerce environment.

Floré Haven represents not only a flower store, but a complete demonstration of front-end and back-end integration, component-based design, and practical application of web development concepts learned throughout the CS50x course.
---

## The Problem It Solves

Many online flower stores lack visual harmony, smooth navigation, or responsive design. Customers often struggle with cluttered layouts, unclear product details, or complicated checkout processes.

Floré Haven addresses these challenges by delivering a clean, organized, and visually refined shopping experience. The platform emphasizes clarity, accessibility, and modern UI principles to make purchasing flowers online simple and enjoyable.

---

## How It Works

The application features a responsive front-end built with React, ensuring compatibility across desktop and mobile devices.

Users can:

- Browse available flower collections
- View detailed product descriptions and pricing
- Add products to their cart
- Manage cart items dynamically
- Complete the checkout process smoothly

The system is structured to prioritize performance, scalability, and maintainable component-based architecture.
